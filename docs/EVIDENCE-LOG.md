Projects the canonical transcript into flattened normalized tokens (each tagged with its owning entry index) using the shared CP2 `normalizeSelectionText`; a passage matches when its normalized token sequence appears as a contiguous run of transcript tokens — no transcript token skipped, entry order never changed, a match may span multiple entries and bridge their timing gaps. Empty/normalization-empty passage, passage longer than available tokens, and a zero-entry transcript each return a frozen empty array without creating a range. A normalization-empty entry between matched words contributes no token, so contiguity holds and the range spans it. Pure, deterministic, immutable (frozen array + candidates + ranges); the canonical transcript is consumed directly — never reparsed or mutated — and non-string passages are rejected via CP2's `TypeError`.
