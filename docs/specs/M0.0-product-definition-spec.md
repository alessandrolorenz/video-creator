# M0.0 — Product Definition Specification

Status: Approved / Frozen — 2026-07-21

## Objective

Freeze the product definition and development method before repository implementation.

## In scope

- vision and personas;
- deterministic transcript-cut scope;
- AI sequence-composer scope;
- MVP and non-goals;
- desktop-first architecture decision;
- domain boundaries;
- export strategy;
- roadmap;
- SDD gates and judges;
- initial risks.

## Out of scope

- repository creation;
- UI implementation;
- model selection benchmark;
- FFmpeg command implementation;
- timeline exporter implementation;
- product naming and branding.

## Acceptance criteria

- product has two clearly separated engines;
- MVP is independently valuable without AI sequence composition;
- React Native is not required for the first product;
- original media remains non-destructive and local-first;
- AI output is typed, validated, and candidate-ID-based;
- professional export has a clear first target and explicit limitations;
- every milestone follows the frozen gate workflow;
- no unresolved decision blocks M0.1 planning.
