Act as an independent judge. Inspect the repository, frozen planning documents, repository audit, and M0.1 implementation plan. Do not trust the plan’s claims without checking them.

Do not modify files.

1. Every in-scope M0.1 requirement maps to concrete work.
2. Every explicit non-goal remains excluded.
3. Renderer has no Node integration or unrestricted privileged API.
4. Context isolation, sandboxing, and typed preload IPC are planned.
5. Runtime input validation exists at IPC boundaries.

8. AI provider SDKs and FFmpeg execution are absent from M0.1.
9. Tests cover time-range invariants, security configuration, and package boundaries.
10. CI reproduces lint, typecheck, tests, and builds.
11. Checkpoints are small and independently reviewable.
12. No unexplained dependency or abstraction is proposed.
13. Worktree preservation and rollback are addressed.
14. The plan stops before implementation.

Return:

- verdict: PASS / PASS WITH NOTES / FAIL;
- evidence table with file/section references;
- blocking issues;
- non-blocking notes;
- scope-creep findings;
